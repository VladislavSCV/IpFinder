Инструкция API для IpFinder

API - это интерфейс программирования приложений, предоставляющий доступ к определенной функциональности. Существуют различные API, например, API для работы с социальными сетями, API для работы с геоданными и т.д. Одним из таких API является API, который работает с IP-адресами.

API для IpFinder позволяет получить информацию о местоположении и других деталях, связанных с определенным IP-адресом.

Адрес запроса

Для того, чтобы сделать запрос к API, нужно отправить GET запрос на URL-адрес:

http://apiFinder/"YOUR IP"

где "YOUR IP" - IP-адрес, данные по которому необходимо получить.

Например, чтобы получить информацию о IP-адресе 20.114.120.146, нужно отправить GET запрос на адрес:

http://apiFinder/20.114.120.146

Ответ API

API для IP возвращает JSON-объект с данными в ответ на GET запрос.

Структура объекта ответа может выглядеть следующим образом:

{
      "ip": "192.168.1.1",
      "country": "Россия",
      "region": "Москва",
      "city": "Москва",
      "timezone": "Europe/Moscow"...
    }
